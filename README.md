# 🌱 小苗成长日记 · AI育儿助手

用AI为孩子建立一份**会思考、会预警**的成长数字档案。  
本项目是为我家小朋友构建的一套智能育儿系统，利用大模型的长链推理与多模态能力，将琐碎的日常记录转化为结构化的成长洞察。

---

## ✨ 功能概览

- **多模态生活记录管道**  
  支持从语音备忘录、随手拍的照片中自动提取关键事件（如第一次独立站立、新学的词语）。
- **儿童健康监测Agent**  
  追踪睡眠、饮食、身高体重，自动绘制个性化生长曲线，对偏离趋势的指标给出温柔预警。
- **成长里程碑推理**  
  基于权威发育量表进行长链推理，判断孩子当前所处的认知/语言/运动节点，并推荐亲子互动方案。
- **专属记忆RAG问答**  
  将所有记录向量化存储，用自然语言随时唤醒珍贵回忆：“宝宝什么时候学会叫妈妈的？”

---

## 🛠️ 技术栈

- **语言**：Python 3.10+
- **AI模型调用**：兼容OpenAI SDK的API（本项目主要针对 **Mimo** 16B-Credit额度设计）
- **向量数据库**：Chroma（轻量级本地存储）
- **文本嵌入**：Mimo Embedding API
- **异步处理**：asyncio
